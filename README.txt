Para compilar basta dar um make em um terminal linux.

========================================

Para executar o programa corretamente deve-se seguir os seguintes passos:

Escreva:

./indexador <estrutura> <buscas> <arquivos[...]>

<estrutura> = Estrutura a ser escolhida: lista, arvore, avl, trie ou hash
<buscas> = Numero de buscas para realizar aleatoriamente na estrutura escolhida
<arquivos> = Nome de todos arquivos separados por espa√ßo

i.e: ./indexador trie 10000 textos/montecarlo.txt
